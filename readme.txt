TicketTool - free event ticketing and managment solution plugin.

== Description ==

TicketTool is events management solution plugin. Using TicketTool.net API it's possible to pull events from your TicketTool.net account and save it to chosen posts category.

Also you can set options to display map, event poster, seating charts, etc.

**Features:**

* Import events from TickeTool.net 
* Add events as posts to chosen category
* Free seating charts, ticket window, build-in google map, event poster, mailchimp integration and more with TicketTool.net.

**TicketTool is excellent solution tested on many blogs.**

== Installation ==

To set the plugin up, you need to:

1. Upload unziped `tickettool` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

That's all!

== Frequently Asked Questions ==

= How to uninstall TicketTool? =
Simple deactivate it through the 'Plugins' menu in Wordpress
